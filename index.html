Brain Tumor Detection using CNN

A Convolutional Neural Network that classifies MRI brain scans into four classes: glioma, meningioma, pituitary, and no tumor. The project uses Kaggle's Brain Tumor MRI dataset and includes careful preprocessing, augmentation, and model evaluation.

Key details
